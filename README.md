#  Simple CRUD-API

### _Before start to work check need to install all dependencies from `package.json` file_

```sh
npm i
```
 To start work in development mode:
 ```sh
npm run start:dev
```
 To start work in production mode:
 ```sh
npm run start:prod
```
 To run tests:
 ```sh
npm run test
```

 To run linter:
 ```sh
npm run lint
```
